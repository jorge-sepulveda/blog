# Personal Website 


## How to run this
In the root folder run:
```sh
hugo serve -t terminal
```

makefile command:
```sh
make local
```

## Share local server(mobile troubleshooting)
```sh
hugo server -t terminal --bind 0.0.0.0 --baseURL http://10.0.0.195:1313/
```

makefile command
```sh
make server
```


## To add new content

```sh
hugo new /path/to/newpost.md
```
